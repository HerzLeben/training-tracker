# 改修要件

今後やりたい改修を、ここに1件ずつ書きためる場所。書いてもらったら、こちらで内容を確認し
実装計画に落として順次対応します。

## 使い方
- 「要件一覧」に `### R-XX タイトル` の形で1件ずつ追記する（番号は R-01, R-02 …）。
- 各項目は下の「記入例」をコピーして埋めればOK。空欄があっても構いません（分かる範囲で）。
- 迷ったら「内容」に日本語で書きたいことを書くだけでも大丈夫です。あとで整理します。

対象画面の呼び方: **Today**（本日のトレーニング）/ **Progress**（進捗）/ **Body**（体組成）/
**Settings**（設定）/ 全体。

優先度: 高 / 中 / 低　　状態: 未着手 / 実装中 / 完了

---

## 記入例（Example）

### R-00 例）Today に前回のメモを表示したい
- 対象画面: Today
- 内容: セッションを開いたとき、前回同じメニューを行ったときのメモ（一言）を上部に表示したい。
- 受け入れ基準: 前回同ワークアウトのメモがあれば表示される。無ければ何も出ない。
- 優先度: 中
- 状態: 未着手
- 補足: メモ機能自体が未実装なら、それも含めて相談。
