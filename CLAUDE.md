## 目的
継続的に目標を達成するための、個人用フィットネス管理アプリ。
体重・体脂肪などを記録し、推移を可視化し、目標に向けたジムの
トレーニングメニューを提案する。

## 形態
- PWA（Progressive Web App）。iPhone のホーム画面に追加してアプリのように使う。
- モバイルファースト（主にスマホから入力する想定）。
- デプロイして実機（iPhone）から利用する。
- **Apple Health 連携は不要。**

## 主な機能
1. 記録：体重・体脂肪率（必要に応じて除脂肪体重・採寸なども）を日々入力。
2. 可視化：推移グラフ（体重・体脂肪の時系列、移動平均など）。
3. 目標管理：目標値（例: 体脂肪率 約15%）と現在地の差分・進捗を表示。
4. メニュー提案：目標（減量／筋肥大／維持）と直近の記録に応じた
   トレーニングメニューを提示。
5. データ所有：記録を CSV / JSON でエクスポートできること。

## 技術方針（提案。Claude Code 側で最適化してよい）
- フロント：React + Vite で PWA 構成（manifest + service worker）。
- データ保存：まずはローカル（IndexedDB 等）。将来的にクラウド同期も検討。
- デプロイ：Vercel / Netlify / Cloudflare Pages のいずれか。
- 将来拡張：
  - プッシュ通知でリマインド（要・送信用サーバー、VAPID 鍵）。
  - メニュー提案を Claude API で動的生成（"AIコーチ"化）。

## ユーザー前提
- 単独利用（本人のみ）。
- Python / R / SQL の経験あり。開発寄りのやり取りは問題なし。

## 最初のゴール（MVP）

「体重・体脂肪を記録 → 推移グラフ → 目標進捗」までが動く最小版。
メニュー提案は次の段階でもよい。
